# Inspección Visual Obligatoria con Playwright

1. Antes de realizar cualquier cambio, debes consultar los lineamientos de diseño en `AGENTS.md`.
2. Una vez realizados los cambios en los componentes Vue o páginas, NUNCA debes darlos por terminados a ciegas.
3. DEBES ejecutar `playwright` (o herramientas equivalentes) para tomar una captura de pantalla del resultado y revisar que la interfaz luzca espectacular.
4. Si la interfaz luce rota, tienes que iterar y arreglarla.
